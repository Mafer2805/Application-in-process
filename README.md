## Video demostrativo

Este repositorio corresponde al proyecto en equipo, cuyo objetivo es desplegar una variable fisiológica en una aplicación móvil. Para ello, se utiliza el sensor MAX30102 conectado a una ESP32, obteniendo los valores de frecuencia cardíaca (BPM) y saturación de oxígeno (SpO₂). Estos datos son enviados mediante comunicación BLE y visualizados en la aplicación desarrollada en Flutter.

En el siguiente video se muestra de dónde se obtienen las señales fisiológicas y cómo se despliegan en el teléfono:
